# kali-additions
This script is just for installing some additional CTF tools.
Simply chmod +x rootflag-install.sh then execute it.
It will create a 'Tools' directory on /root/ and install all the tools.

---
## Tools included:

- Atom editor
- Terminator
- LinPeas
- LinEnum
- pspy32 & pspy64
- Sysinternals suite
- John tools
- Evil winrm
- Bloodhound
- Evil-WinRAR
- Patator
- Netcat binaries
- ghidra
- OhMyZsh Shell
